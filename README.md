# Gandhi Mithal - Software Assistance Website

A professional website providing software assistance with an integrated Q&A feature.

## Local Viewing
Simply open `index.html` in any web browser.

## Deployment Options for Live Website Link

### Option 1: GitHub Pages (Free & Recommended)
1. Create a GitHub account if you don't have one
2. Create a new repository named `gandhi-mithal-software`
3. Upload these files to the repository
4. Go to Settings > Pages
5. Select main branch as source
6. Your site will be live at: `https://yourusername.github.io/gandhi-mithal-software`

### Option 2: Netlify (Free, Easiest)
1. Go to https://www.netlify.com
2. Sign up for free
3. Drag and drop the entire `Gandhi` folder
4. Your site will be live instantly at: `https://your-site-name.netlify.app`
5. You can customize the domain name

### Option 3: Vercel (Free)
1. Go to https://vercel.com
2. Sign up for free
3. Import your project
4. Deploy with one click
5. Get a live URL: `https://your-site-name.vercel.app`

### Option 4: Custom Domain
- Purchase a domain from GoDaddy, Namecheap, or Google Domains
- Use any hosting provider (Netlify, Vercel, or traditional hosting)
- Point your domain to the hosting service

## Features
- Responsive design (mobile and desktop)
- Search functionality for Q&A
- Professional gradient theme
- Easy to add more Q&A entries in `script.js`

## Adding More Q&A Entries
Edit the `qaDatabase` array in `script.js`:

```javascript
{
    question: "Your question here",
    answer: "Your answer here",
    keywords: ["keyword1", "keyword2", "keyword3"]
}
```
